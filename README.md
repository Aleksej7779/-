# -
Данный учебный проект посвящен разработке и внедрению решений по доработке или замене существующих информационных систем. Проект направлен на формирование навыков анализа текущих систем, выявления их недостатков и разработки улучшенных решений, которые соответствуют современным требованиям бизнеса и технологий.
